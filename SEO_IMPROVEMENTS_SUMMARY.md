## 🔧 Технические детали

- **Framework**: React 18.3.1 + Vite + React Router v6
- **UI Components**: shadcn/ui (Radix UI)
- **Форма**: Native HTML inputs с React state для маски
- **Стили**: Tailwind CSS + Radix Colors

## 📝 Примечания

- React-markdown не установлена (не требуется, используется JSX)
- Все формы готовы к интеграции с backend'ом
- Телефонная маска жёстко настроена на формат +7 (XXX) XXX-XX-XX
- lastModified дата установлена на 2026-05-09
- Все страницы доступны и полностью функциональны
